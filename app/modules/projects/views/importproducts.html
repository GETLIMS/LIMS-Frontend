To import a list of products two files are needed: a TSV (tab seperated) text file that contains information on the products to be imported and (optionaly) a zip file containing the design files.

The following headers are valid for use in the product information file. Those marked with an asterix (*) are required: name*, status*, product_type*, optimised_for.

All information supplied must match a valid input for the corrosponding field or it will be rejected. Some values are case sensitive, using the incorrect case will cause the entry to be rejected.
